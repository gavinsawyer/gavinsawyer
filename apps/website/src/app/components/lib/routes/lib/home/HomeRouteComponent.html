A free and open-source API and library of iOS shortcuts used to create highly detailed home automations.

Automations in Apple’s HomeKit system run on tvOS devices such as Apple TV and HomePod. This makes it impossible to reference the user's Focus mode and therefore to do things such as disable motion-activated lights when in Sleep focus.

This project addresses that limitation and allows for much more.
